Gebruiker worden

Door klant te worden bij Medispring sluit u zich aan bij een gemeenschap van zorgverleners samengebracht in een coöperatie.
Uw bijdrage wordt als eerste geherinvesteerd in de software, de ondersteuning, trainings en nieuwe functionaliteiten en zal altijd hergebruikt worden voor de coöperatie en zorgverleners.

De volgende stappen

Eenmaal het inschrijvingsformulier is ingevuld zijn er nog een aantal stappen te volgen alvorens u dagelijks met onze software kan beginnen werken :

1

Coöperant worden

Medispring is een coöperatieve vennootschap. Om de software van Medispring te gebruiken wordt u uitgenodigd om coöperant te worden via de volgende site: Coopérative Medispring. We verzoeken onze toekomstige coöperaten volgende informatienota la note d’information relative à la coopérative door te nemen alvorens de inschrijvingsprocedure te starten. 
2

Technische formaliteiten (eHealth + gezondheidsnetwerk)

Indien nodig helpen wij u graag verder bij het aanvragen van het eHealth-certificaat en/of uw inschrijving bij een gezondheidsnetwerk.
3

Installatie van de software

Installation du logiciel sur votre ordinateur, à distance via TeamViewer, par nos équipes. Notre logiciel requiert de tourner sur un matériel informatique avec les caractéristiques MINIMALES suivantes : un processeur de minimum Core i5 8ième génération, un disque dur de type SSD avec plus de 250Gb, avec 8 Go de RAM. Naturellement, votre expérience dépendra de l'utilisation que vous ferez en parallèle de notre logiciel : une configuration plus puissante est toujours utile. Installatie van de software op uw computer, op afstand via TeamViewer, door onze teams. Onze software moet draaien op computerhardware met de volgende MINIMALE kenmerken: een minimale Core i5 8e generatie processor, een SSD type harde schijf met meer dan 250Gb, met 8 GB RAM. Uw ervaring hangt natuurlijk af van hoe u onze software parallel gebruikt: een krachtigere configuratie is altijd handig.
4

Training

U krijgt van ons, vanop afstand, een opleiding omtrent de software zodat u het hele systeem leert kennen.
